MatrixMaster Interactive Tutorial

MatrixMasterの強力な機能を体験してください。以下の「作例をロード」ボタンを押すと、計算機にサンプルデータが自動で入力されます。

3x3 記号行列式

余因子展開アルゴリズムを用いて、文字を含んだn×n行列の行列式を導出します。

A = [[a, b, c], [d, e, f], [g, h, i]]

2x2 記号逆行列

余因子行列を用いて、逆行列を記号のまま計算します。数式は `A^{-1} = \frac{1}{\det(A)} \text{adj}(A)`

A = [[a, b], [c, d]]

固有値問題 (特性多項式)

固有値 `\lambda` を求めるための方程式 `\det(A - \lambda I) = 0` の左辺を計算します。

A = [[1, 2], [3, 4]]

連立一次方程式

`Ax=b` を解きます。作例をロードすると、自動的に方程式ソルバーページに移動します。

A = [[2,1,-1],[ -3,-1,2],[-2,1,2]]
b = [8,-11,-3]
